# 마! 내가 너 남자친구다.

폴더 별 기능

```
boyfriend-chatbot : 남자친구가 대화(채팅)하는 인공지능 데이터 셋을 학습하는 코드
boyfriend-voice : 남자친구가 대화(음성)하는 인공지능 데이터 셋을 학습하는 코드

collection-from-telegram : 텔레그램의 봇을 통한 학습용 데이터 생성 프로그램 (채팅 데이터만 데이터베이스에 실시간으로 저장함.)
collection-to-csv : 텔레그램 봇으로 부터 생성 된 데이터를 학습하기 위한 CSV 파일 형태로 변환하는 프로그램

client-for-boyfriend : 추론 서버와 통신하는 임베디드 프로그램 (현재 On-Device Inference는 수행하지 않음)
```
